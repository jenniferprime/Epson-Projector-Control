# Epson Projector Control
A lil c# thing that controls Epson Projectors

# KeyCodes (EB-96W)
      3B = Powerkey (Send twice for poweroff)
      67 = Source Search
      43 = Source Computer
      46 = Source Video
      85 = Source USB
      8A = Source LAN
      3E = AV Mute
      47 = Freeze
      57 = Volume Down
      56 = Volume Up
      68 = Page Up
      96 = Page Down
      
# Known working Projectors
  Epson EB-96W
